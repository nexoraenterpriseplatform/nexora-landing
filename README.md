# NEXORA Official Website

Website oficial da **NEXORA Enterprise Platform**.

## Objetivo

Apresentar a visão da NEXORA, captar interessados para o Programa Beta e preparar a presença pública da marca.

A Landing Page é a primeira experiência pública da NEXORA e servirá como base para a evolução futura do **NEXORA Official Website**.
