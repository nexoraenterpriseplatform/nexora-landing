# Release 0.9.2 — Programa Beta Foundation

## Objetivo

Preparar o formulário do Programa Beta para integração com Google Sheets.

## Adicionado

- Arquivo `js/config.js` para configurar a URL do Google Apps Script.
- Validação de campos obrigatórios no formulário.
- Validação de e-mail.
- Estado de carregamento no botão de envio.
- Mensagem de sucesso.
- Mensagem de erro.
- Documento `docs/Google-Sheets-Integration.md` com passo a passo de integração.

## Observação

A integração real com Google Sheets depende da criação da planilha e publicação do Apps Script.
Enquanto a URL não for configurada, o formulário valida os dados e mostra mensagem de integração pendente.
