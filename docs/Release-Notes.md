# Release Notes — NEXORA Official Website

## Release 0.9.3 — Production Readiness

**Objetivo:** preparar a Landing para ambiente de produção antes da integração final e publicação.

### Entregas

- SEO Foundation implementado.
- Open Graph e Twitter Cards configurados.
- `robots.txt`, `sitemap.xml`, `site.webmanifest` e `browserconfig.xml` criados.
- Favicon e ícones base gerados.
- JSON-LD institucional adicionado.
- Formulário Beta recebeu melhorias de acessibilidade.
- Mensagens de sucesso/erro refinadas visualmente.
- Documentação e CHANGELOG atualizados.

### Checklist de validação

- Abrir `index.html` localmente.
- Pressionar `Ctrl + F5`.
- Conferir se a Landing abre normalmente.
- Testar envio vazio do formulário.
- Testar envio preenchido.
- Conferir se o favicon aparece na aba do navegador.
- Conferir se o Footer aparece no final.
